# Weather Web Application

A simple and responsive **Weather Web Application** built using  
**HTML, CSS, and JavaScript**.  
This app fetches real-time weather data using the **OpenWeatherMap API**  
based on user-entered city name or the user’s current location.

---

##  Features

- Get weather by **current location**
-  Search weather by **city name**
-  Displays:
  - Temperature (°C)
  - Weather condition
  - Humidity
  - Wind speed
-  Responsive & user-friendly UI

---

##  Technologies Used

- HTML5  
- CSS3  
- JavaScript (Fetch API)  
- OpenWeatherMap API
